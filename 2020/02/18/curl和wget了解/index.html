字数总计: 241|阅读时长: 1 分钟

curl

curl是系统自带的c库，简单过一下，记录一下参数。
不带参数就是，GET请求。

-A

指定客户端的代理标头

-B

发送cookie

-X

用指定方式请求

-d

用Post请求

-L

跟随服务器重定向

-H

设置http请求头信息

-k

跳过ssl检测(https)

-O

以远程的文件名下载保存

-o

以自定的文件名下载保存

-s

不显示无关信息

-v

显示所有信息

wget

系统自带的下载文件的工具。

-O

以指定文件名存文件

–limit-rate=

限速

-c

断点续传

-b

后台下载

-U

设置user-agent

-mirror

镜像某个网站

-p

下载网页的所有内容

-r

递归下载所有网页中所有链接
