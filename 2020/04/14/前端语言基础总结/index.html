    2020-04-14
    字数总计: 1.6k|阅读时长: 7 分钟

    HTML

    基本元素

    图像 img标签

    需要设置其属性 src指定图像的路径；
    width=“200” height="200"设置大小；

    img不能够自己居中，需要放在其他能够居中的标签中实现这个效果，比如h1标签,p标签.
    经常采用的手段是放在div中居中

    超链接 a标签

    表格 table标签

    属性

    属性border： 表形式
    属性 width
    属性align： 元素位置
    设置tr或者td的属性bgcolor： 背景色

    列表 ul标签和ol标签

    列表分无序列表和有序列表
    分别用

      标签和
        标签表示

    1. 无序列表
    2. 有序列表

    块标签 div和span

    这两种标签都是布局用的
    这种标签本身没有任何显示效果
    通常是用来结合css进行页面布局

    两者区别

    div是块元素，即自动换行
    常见的块元素还有h1,table,p
    span是内联元素，即不会换行
    常见的内联元素还有img,a,b,strong

    fron字体标签

    可以调整颜色color
    大小size

    内联框架 iframe

    通过内联框架 可以实现在网页中“插入”网页

    表单元素

    表单 form

    向服务器提交数据, action表示将数据提交到哪里，method用来选择http提交数据方式

    input类

    使用type选择对应的类型

    文本框
    密码框 t
    选框

    单选：input type=“radio”
    复选：input type=“checkbox”
    checked默认选

    下拉列表 select

    使用select和option, size框的大小，multiple表示可以多选，selected默认选

    普通按钮button

    value显示文本，不能用于提交数据。

    提交按钮submit
    重置按钮reset

    将输入框复原

    图像提交 input image

    src放图片uri

    button按钮标签

    CSS

    可以控制内容的属性，比如颜色等。

    格式 style框住

    选择器{属性：值}

    选择器

    • id选择器： #p 每个标签的id都应该是唯一的
    • 类选择器： …pre{color:blue;}
    • 准确选择: 根据元素名和class选择。

    CSS与html文件分离

    创建一个文件叫style.css
    其内容为

    .p1{
    color:red;
    }
    .span1{
    color:blue;
    }

    然后在html中包含该文件
